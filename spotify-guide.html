Installation du DNS (Étape Importante)

Avant d’installer Spotify, il est important de configurer un DNS dédié pour éviter les blocages d’Apple, qui applique des restrictions strictes..

Installer le DNS

Comment installer le DNS en quelques étapes simples :

  1. Ouvre Réglages sur ton iPhone.
  2. Va dans Général puis VPN et gestion de l'appareil.
  3. Sélectionne le profil DNS se nommant "kdoidvn".
  4. Appuie sur Installer en haut à droite.
